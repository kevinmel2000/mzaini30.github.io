# Situs Offline Muhammad Zaini

## Mengunduh Isi Situs

1. Klik `Clone or Download`
2. Klik `Download ZIP`

## Menjalankan Situs

### Menggunakan Jekyll

1. Buka Terminal lalu ketikkan `jekyll serve`

### Menggunakan Python

1. Buka `_site`
2. Buka Terminal lalu ketikkan `python -m SimpleHTTPServer`
